# F1 Standings Archive

A web app for viewing Formula 1 driver and constructor standings from different seasons.

## Features

- Select a Formula 1 season
- View driver standings
- View constructor standings
- Compare teams and drivers by points

## Tech Stack

- React
- Vite
- TypeScript
- Express.js
- Sqlite

## How to Run

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
